# Open-Source Hurricane Imagery Tool
A work-in-progress tool designed to streamline the process of retrieving satellite imagery for past tropical cyclones.

### Current Features
1. Retrieve Band 9 and 13 satellite data from the GOES-R series, as far back as 2017. 
2. Plot satellite imagery of a tropical cyclone at its peak intensity (peak winds as of now)
3. Plot the entire lifespan of a tropical cyclone
